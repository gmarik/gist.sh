h2. About

@gist.sh@ is a simple bash script for getting/posting code snippets (gists) from/to http://gist.github.com

h2. Installation 

<pre>
  $ curl http://github.com/gmarik/gist.sh/raw/master/gist.sh > gist.sh && 
    chmod 755 gist.sh && 
    sudo mv gist.sh /usr/local/bin/ 
</pre>


h2. Usage

* Posting to GitHub:

<pre>
  $ cat file | gist.sh
</pre>

or

<pre>
  $ gist.sh < file
</pre>

* Getting from GitHub:

<pre>
  $ gist.sh 1234
</pre>
